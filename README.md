# 🥏 Générateur de Discours - Ultimate Frisbee

Une application web simple et moderne pour générer des discours motivants pour vos matches d'Ultimate Frisbee.

## ✨ Caractéristiques

- ✅ Application 100% front-end (HTML, CSS, JavaScript)
- ✅ Responsive et optimisée pour les smartphones
- ✅ Génère automatiquement 6 phrases aléatoires
- ✅ 3 catégories de phrases :
  - Esprit du jeu
  - Aspect sportif
  - Remerciements
- ✅ Design moderne et épuré
- ✅ Compatible avec GitHub Pages
- ✅ Aucune dépendance externe

## 📁 Structure du projet

```
HuddleGenerator/

```

## 🚀 Installation et utilisation

### En local

1. Clonez ou téléchargez le projet
2. Ouvrez `index.html` dans votre navigateur web
3. Cliquez sur le bouton "Générer un Discours" pour créer vos 6 phrases

### Sur GitHub Pages

1. Créez un nouveau repository public sur GitHub
2. Poussez le contenu de ce projet dans le repository
3. Allez dans les paramètres du repository → Pages
4. Sélectionnez `main` (ou `master`) comme branche source
5. L'application sera disponible à : `https://votreusername.github.io/nomdurepo`

## 📝 Modifier les phrases

Pour ajouter ou modifier les phrases :

1. Ouvrez le fichier correspondant :
   - `phrases-spirit.js` pour l'esprit du jeu
   - `phrases-sport.js` pour l'aspect sportif
   - `phrases-thanks.js` pour les remerciements

2. Ajoutez ou modifiez les phrases dans le tableau `spiritPhrases`, `sportPhrases` ou `thanksPhrases`

3. Les changements prendront effet immédiatement
